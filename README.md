# THISCREEN

THISCREEN es una aplicación de escritorio para macOS que permite capturar y grabar la pantalla con herramientas de edición avanzadas. Diseñada para usuarios que necesitan una solución rápida y potente para tomar capturas y grabaciones con anotaciones visuales integradas.

## Características

- **Captura de pantalla**: Toma capturas de pantalla seleccionando un área específica.
- **Grabación de pantalla**: Graba secciones seleccionadas o la pantalla completa.
- **Edición visual**: Anota tus capturas y grabaciones con herramientas de dibujo como lápiz, flechas, formas y texto.
- **Edición avanzada**: Pixela secciones sensibles, recorta áreas específicas y aplica zoom.
- **Atajos de teclado**: Accede rápidamente a funciones clave usando atajos como ⌘⇧S (captura) o ⌘⇧R (grabación).

## Cómo usar

1. **Capturar pantalla**:
   - Presiona `⌘⇧S` o haz clic en el botón de captura en la barra de estado.
   - Selecciona el área de la pantalla que deseas capturar.
2. **Editar contenido**:
   - Usa las herramientas de dibujo en la parte inferior para anotar.
   - Añade texto, formas, líneas, etc.
3. **Guardar contenido**:
   - Presiona el botón de guardar (✓) para guardar en una ubicación predeterminada o personalizada.

## Atajos de teclado

- `⌘⇧S`: Tomar captura de pantalla
- `⌘⇧A`: Grabar pantalla completa
- `⌘⇧R`: Grabar área seleccionada
- `⌘S`: Guardar captura/grabación
- `⌘D`: Guardar en Escritorio
- `⌘L`: Guardar en Descargas
- `⌘Z`: Deshacer última acción
- `⌘C`: Copiar al portapapeles

## Requisitos del sistema

- macOS 12.0 o superior
